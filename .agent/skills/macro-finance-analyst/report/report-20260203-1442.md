## ⚠️ 风险提示

> [!CAUTION]
> **流动性危机陷阱**
> 
> 当 RRP 归零 + TGA 持续抽水时，不要误判为"利好避险资产"。历史教训：
> - **2008年金融危机**: 黄金暴跌
> - **2020年3月**: 黄金也跌 (Dash for Cash)
> 
> 此时所有资产都是"提款机"，而非"避风港"。

### 需持续监控的关键变量

1. **TGA 变化**: 若开始下降 → 利好黄金
2. **银行准备金**: 若回升至 3.0万亿以上 → 流动性改善
3. **DGS2 走势**: 若继续下降 → 降息预期强化，利好黄金
4. **MOVE 指数**: 若突破 100 → 高度警惕
